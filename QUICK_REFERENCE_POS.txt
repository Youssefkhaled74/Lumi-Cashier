┌─────────────────────────────────────────────────────────────────────┐
│ 🐛 COMMON ISSUES & FIXES                                            │
└─────────────────────────────────────────────────────────────────────┘

Language not switching:
→ Clear: php artisan config:clear
→ Clear: localStorage.clear() in browser

Cart not saving:
→ Check: localStorage quota
→ Inspect: localStorage.getItem('posCart')

Receipt not printing:
→ Use: Chrome or Edge browser
→ Check: Printer settings (80mm width)
→ Verify: @media print styles loaded

RTL layout broken:
→ Check: <html dir="rtl"> in source
→ Verify: Tajawal font loaded
→ Inspect: [dir="rtl"] CSS selectors

┌─────────────────────────────────────────────────────────────────────┐
│ 🔄 TYPICAL WORKFLOW                                                 │
└─────────────────────────────────────────────────────────────────────┘

1. Cashier logs in
2. Opens POS (/admin/pos)
3. Switches language if needed (🌐)
4. Searches product (F2)
5. Adds to cart (click + or Quick Add)
6. Adjusts quantities (+/-)
7. Clicks "Proceed to Checkout" (F4)
8. Selects payment method
9. Adds notes (optional)
10. Clicks "Complete Payment"
11. System creates order
12. Prints receipt (🖨️)
13. Ready for next customer!
